# 🛡️ Fact-Check Agent

AI-powered PDF fact-checking web application built using Streamlit and local LLMs via Ollama.

The system automatically:
- extracts factual claims from uploaded PDFs
- identifies statistics and measurable statements
- verifies claims using AI reasoning
- flags misinformation and inaccurate data
- generates downloadable fact-check reports

---

# 🚀 Features

✅ Upload PDF documents  
✅ Extract factual/statistical claims  
✅ AI-powered claim verification  
✅ Detect misinformation  
✅ Downloadable JSON reports  
✅ Minimal clean UI  
✅ Local AI using Ollama  
✅ No paid API required  

---

# 🧠 Tech Stack

- Ollama
